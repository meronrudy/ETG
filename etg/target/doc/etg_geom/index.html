G-ETG dynamic geometry: time-varying metric/speed fields and curvature interface.

This crate introduces a time-indexed geometry API suitable for slice-local reasoning and invariance tests. It provides:

  • GeometryT(): slice-local distance and invariant speed accessors.
  • ScaledEuclidDyn(): a 1D Euclidean geometry with time-varying spatial scale and speed.
  • Curvature(): a minimal curvature interface with an Ollivier–Ricci-like edge kappa.

Examples

use etg_core::Scalar;
use etg_geom::{GeometryT, ScaledEuclidDyn};

let geom = ScaledEuclidDyn {
    scale: |t: Scalar| 1.0 + 0.1 * t,
    speed: |t: Scalar| 1.0 + 0.2 * t,
};

let t0: Scalar = 0.3;
let d = geom.dist_at(t0, &(-1.0), &(2.0));
let c = geom.c_at(t0);
assert!(d > 0.0);
assert!(c > 0.0);
